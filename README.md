# print_color

simply organize debugging

![screenshot](screenshot.png)


# usage
* import `import 'package:print_color/print_color.dart';`
* just `Print.green('someText')`


# example
```dart
import 'package:print_color/print_color.dart';

Print.white('white');
Print.black('black');
Print.cyan('cyan');
Print.blue('blue');
Print.green('green');
Print.red('red');
Print.magenta('magenta');
Print.yellow('yellow');
```
