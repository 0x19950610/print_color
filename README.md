# print_color

print(), in colors

![screenshot](screenshot.png)

## usage

* `import 'package:print_color/print_color.dart';`
* `Print.green('someText');`

## example

```dart
import 'package:print_color/print_color.dart';

Print.white('white');
Print.black('black');
Print.cyan('cyan');
Print.blue('blue');
Print.green('green');
Print.red('red');
Print.magenta('magenta');
Print.yellow('yellow');
```

## colors

* white
* black
* cyan
* blue
* green
* red
* magenta
* yellow
